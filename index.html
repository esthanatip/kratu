ดึงข้อมูลจาก Google Sheets · รองรับทั้ง สส. และ สว.

⚙️ ตั้งค่าการเชื่อมต่อ ▼

✂️ วิธีง่ายที่สุด: คัดลอกข้อมูลจาก Google Sheets แล้ววางที่นี่
① เปิด Sheet → Ctrl+A เลือกทั้งหมด → Ctrl+C
② วางในช่องด้านล่าง → กด ใช้ข้อมูลนี้

🔗 วิธี Apps Script (เสถียรที่สุด):
① Google Sheets → ส่วนขยาย → Apps Script → วางโค้ด → บันทึก
② Deploy → New deployment → Web app → Anyone → Deploy → คัดลอก URL
③ นำ URL มาวางด้านล่าง → กด โหลดข้อมูล
🗂️ ลำดับคอลัมน์ใน Sheet (A=1, B=2 ...)

💡 เลื่อน=เหลือง · ตอบชี้แจง/ตอบเรียบร้อย=เขียว · ค้าง=แดง (จากคอลัมน์คำตอบ)
