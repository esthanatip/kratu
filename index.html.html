ระบบแสดงกระทู้ถามในที่ประชุมสภา

ดึงข้อมูลจาก Google Sheets · รองรับทั้ง สส. และ สว.

⚙️ ตั้งค่าการเชื่อมต่อ Google Sheets ▲
🏆 วิธีที่แนะนำ — เสถียร 100% ไม่พึ่ง proxy
① เปิด Google Sheets → เมนู ส่วนขยาย → Apps Script
② วางโค้ดด้านล่างทั้งหมด แล้วกด 💾 บันทึก
③ คลิก Deploy → New deployment → Web app
④ ตั้ง "Who has access" เป็น Anyone → Deploy → คัดลอก URL
⑤ นำ URL มาวางในช่อง "URL" ด้านล่าง แล้วคลิก เชื่อมต่อ
📋 วิธีทั่วไป (อาจมีปัญหา CORS บ้าง):
① เปิด Sheet → ไฟล์ → แชร์ → เผยแพร่ไปยังเว็บ → เลือก CSV → คัดลอก URL แล้วนำมาวางด้านล่าง
② หรือตั้ง Sheet เป็น "ทุกคนที่มีลิงก์สามารถดู" แล้วใช้รูปแบบ:
https://docs.google.com/spreadsheets/d/SHEET_ID/export?format=csv&gid=GID
③ แถวแรกของ Sheet ควรเป็น หัวคอลัมน์ (จะถูกข้ามอัตโนมัติ)
✂️ วิธีที่ง่ายที่สุด: คัดลอก-วาง CSV จาก Google Sheets โดยตรง (ไม่ต้องพึ่ง URL)

① เปิด Google Sheets → ไฟล์ → ดาวน์โหลด → CSV แล้วเปิดไฟล์ด้วย Notepad/TextEdit
② หรือใน Google Sheets กด Ctrl+A เลือกทั้งหมด → Ctrl+C คัดลอก
③ วางในช่องด้านล่างแล้วคลิก "ใช้ข้อมูลนี้"

🗂️ ลำดับคอลัมน์ใน Sheet (A=1, B=2, C=3 ...)

💡 สีไฮไลต์กำหนดอัตโนมัติจากคอลัมน์ "คำตอบ": คำว่า เลื่อน = เหลือง · ตอบชี้แจง / ตอบเรียบร้อย = เขียว · ค้าง = แดง
